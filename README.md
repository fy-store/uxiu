# 项目简介

该项目是为 koa 后端应用提供基础模块的工具包

## 快速开始

使用 cli 工具快速搭建项目骨架

**全局安装**

**pnpm**

```bash
pnpm i -g uxiu-cli
```

**yarn**

```bash
yarn global add uxiu-cli
```

**npm**

```bash
npm i -g uxiu-cli
```

**临时运行 cli**

**pnpm**

```bash
pnpx uxiu-cli
```

**yarn**

```bash
yarn dlx uxiu-cli
```

**npm**

```bash
npx uxiu-cli
```
